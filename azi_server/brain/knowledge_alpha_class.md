# Alpha Craft Class - Ürün Bilgisi (AZI Knowledge)
**Ürün Kodu:** PROD-CLASS-ENT
**Tür:** Eğitim Yönetim Sistemi (LMS)

## Tanım
Alpha Craft Class, dershaneler ve kurs merkezleri için geliştirilmiş, yapay zeka destekli bir sınıf ve yoklama yönetim platformudur. Ödeme takibi içermez, tamamen akademik veri ve operasyona odaklanır.

## Temel Özellikler
1. **Sınıf Yönetimi:**
   - Esnek ders planlama (Saat, süre, gün seçimi).
   - "Yerleştirmeli" ve "Seviye Bazlı" sınıf atamaları.

2. **Yoklama & Devamsızlık (AI Powered):**
   - Günlük hızlı yoklama modu.
   - Devamsızlık sınırını aşan öğrencileri AZI otomatik tespit eder.
   - Veli arama ve not ekleme önerileri sunar.

3. **Mekan Analizi:**
   - Binanın kuşbakışı doluluk haritası.
   - Canlı ders takibi (Yanıp sönen sınıflar).

4. **Kullanıcı Komutları:**
   - "Sınıf oluştur" -> Yeni sınıf wizardını tetikler.
   - "Yoklama al" -> Günlük yoklama ekranını açar.
   - "Riskli öğrenci var mı?" -> Devamsızlık raporunu sunar.

## Teknik Detaylar
- **Veritabanı:** JSON tabanlı yerel veritabanı (Cloud Sync destekli).
- **Entegrasyon:** AZI Server üzerinden lisans kontrolü yapar.
- **Platform:** Windows (Exe).

## AZI İçin Notlar
- Kullanıcı "sınıf programı" veya "dershane yazılımı" dediğinde bu üründen bahsediyordur.
- Devamsızlık sorularında "Risk Analizi" modülünü referans göster.
